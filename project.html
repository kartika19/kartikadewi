Aplikasi Analisis Kondisi Kesehatan Koperasi

-Project yang telah saya kerjakan -

Saya membuat sebuah Aplikasi Analisis Kondisi Kesehatan Koperasi berbasis Website bernama KESKOP.

Aplikasi ini dapat menampilkan kondisi kesehatan koperasi pada periode tertentu. Selain itu, aplikasi ini juga dapat menampilkan rekomendasi perbaikan yang dapat dilakukan oleh koperasi untuk mempertahankan predikatnya

Dalam membuat website ini saya menggunakan HTML, PHP, JavaScript dan Bootstrap serta database MySql

Halaman Admin

Halaman ini merupakan dashboard yang digunakan oleh admin untuk memantau penilaian kesehatan koperasi

Halaman Master Data Aspek

Halaman ini digunakan untuk mengelola data aspek seperti menambah, mengubah atau menghapus data aspek

Halaman Master Data Saran

Halaman ini digunakan untuk menambah, mengubah atau menghapus saran

Halaman Input Data Penilaian

Halaman ini digunakan untuk menambahkan data penilaian koperasi

Laporan Penilaian Kondisi Kesehatan Koperasi

Halaman ini berisi laporan penilaian kondisi kesehatan koperasi pada periode tertentu

Halaman Saran dan Rekomendasi

Halaman ini berisi saran dan rekomendasi perbaikan yang dapat dilakukan oleh koperasi untuk memperbaiki aspek yang belum maksimal
