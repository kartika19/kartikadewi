Website Toko Baju Online

-Project yang telah saya kerjakan -

Saya membuat sebuah Website Toko Baju Online.

Dengan menggunakan website ini, pengguna dapat melihat hingga memesan produk dari toko ini. Selain itu admin juga dapat mengelola produk hingga pesanan yang dilakukan pelanggan.

Dalam membuat website ini saya menggunakan HTML, PHP, JavaScript dan Bootstrap serta database MySql

Halaman Produk

Halaman ini digunakan untuk melihat jenis-jenis produk yang tersedia, selain itu, user juga dapat memesan produl dengan menekan tombol pesan.

Halaman Keranjang

Halaman ini merupakan halaman untuk menampilkan produk-produk yang telah dipilih sebelumnya oleh user. Jika ingin memesan, user tinggal menekan tombol checkout, maka produk tersebut akan dipindahkan ke halaman checkout.

Halaman Checkout

Halaman ini digunakan sebagai validasi jika ingin memesan dan melakukan pembayaran dengan menekan tombol lnajut pembayaran

Halaman Metode Pembayaran

Halaman ini digunakan untuk memilih metode pembayaran

Halaman Pembayaran

Halaman ini digunakan untuk melakukan pembayaran dengan memasukkan bukti transaksi.

Halaman Akun

Halaman ini berisi informasi user yang login serta informasi pesanan yang sudah dilakukan.

Halaman Admin

Halaman ini berisi informasi-informasi rinci produk hingga pemesanan.

Halaman Produk (Admin)

Halaman ini berisi data-data produk serta dapat mengelola produk seperti menambah, mengubah dan menghapus produk.

Halaman Pesanan

Halaman ini digunakan untuk mengelola pesanan, admin dapat mengubah status atau menolak pesanan.
